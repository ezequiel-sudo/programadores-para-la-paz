Al enviar una petición POST a la ruta http://localhost:3000/registro con el JSON de María, el servidor Express procesa con éxito el cuerpo de la solicitud gracias al middleware express.json(). 

El servidor devuelve una respuesta en formato JSON con un código de estado 200 (OK). El objeto retornado incluye la propiedad "estado": "Datos recibidos" junto con los valores originales de "nombre" y "mensaje" que fueron extraídos dinámicamente desde el objeto req.body. Esto confirma que el servidor interceptó, leyó y procesó la información del cliente de manera correcta.
